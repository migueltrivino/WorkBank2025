Workbank Documentation
======================

Bienvenido a la documentación oficial del proyecto **Workbank 2025**.  
Aquí encontrarás información sobre instalación, arquitectura del backend y frontend,
uso de la API y manual de usuario.

:Versión: 1.0  
:Fecha: Octubre 2025  
:Autor: Equipo de Desarrollo Work Bank  
:Proyecto: Work Bank - Sitio web de trabajos remunerados  

-----------------------------------------------------------
1. Introducción
-----------------------------------------------------------

El presente manual de usuario tiene como propósito guiar al usuario en el acceso e interacción con las funciones principales de la plataforma **Work Bank**, enfocándose en los módulos de **Inicio de Sesión (Login)** y **Página Principal (Index)**.

Work Bank es una aplicación web diseñada para conectar **trabajadores independientes** con **clientes** que requieren servicios específicos.  
La plataforma permite publicar, buscar y gestionar ofertas laborales, así como administrar la información de los usuarios y servicios disponibles.

-----------------------------------------------------------
2. Público Objetivo
-----------------------------------------------------------

Este manual está dirigido a:

- Usuarios nuevos que desean acceder a la plataforma por primera vez.  
- Trabajadores que buscan publicar o gestionar sus ofertas de servicio.  
- Clientes que buscan contratar servicios a través del buscador principal.  
- Administradores del sistema que gestionan usuarios, servicios y ofertas.

-----------------------------------------------------------
3. Requisitos del Sistema
-----------------------------------------------------------

Para garantizar el correcto funcionamiento de la aplicación, se recomienda cumplir los siguientes requisitos mínimos:

**Requisitos de software**
  - Navegador web actualizado (Google Chrome, Mozilla Firefox, Edge).  
  - Conexión estable a internet.  
  - Acceso al dominio o dirección local del proyecto (por ejemplo: ``http://localhost:5173``).

**Requisitos de acceso**
  - Contar con un usuario y contraseña válidos.  
  - En caso de no tener cuenta, el usuario podrá registrarse mediante el formulario de registro.
